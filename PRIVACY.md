# Privacy notice

Spot Ambience does not create user accounts and does not collect analytics,
advertising identifiers, contacts, messages, location, or personal profiles.

The application makes HTTPS requests to `dashboard.elering.ee`. Spot price
data: Nord Pool day-ahead prices for Finland, published through Elering LIVE by
Elering AS. As with any internet request, the service can receive normal
network metadata such as an IP address. No account credentials or personal
application data are sent.

Price history, settings, selected ambience URLs, and runtime status are stored
locally on the phone. They are not uploaded by Spot Ambience. The application
does not read or transmit unrelated user files.

The application does not provide a public data API, bulk export, or any onward
transmission of the cached price data. Prices may be delayed, corrected,
unavailable, or inaccurate; the application is provided as-is.
