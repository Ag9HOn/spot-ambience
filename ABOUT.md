# About Spot Ambience

Spot Ambience shows Finnish quarter-hour electricity prices and optionally
changes the Sailfish ambience to reflect the average price of the next hour.
It is a free community application for Sailfish OS.

Package release: 1.0.0-1. Source and release information are available through the
project repository.

Original work created by Petteri Nakamura.
https://www.linkedin.com/in/petterinakamura/

Project source: https://github.com/Ag9HOn/spot-ambience

Spot price data: Nord Pool day-ahead prices for Finland, published through
Elering LIVE by Elering AS. Elering is Estonia's state-owned electricity and
gas transmission-system operator. The app stores a local cache and does not
provide a public data API or bulk export. It is not affiliated with Elering or
Nord Pool.
